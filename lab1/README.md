## Install GHDL and GTKWave:
- Installed Notepad++
- Downloaded GHDL
- Downloaded GTKWave
- Added to Environment Variables Path

Entered into terminal:
> $ git clone https://github.com/kevinwlu/dsd.git
$ mkdir vhdl
$ cd vhdl
$ cp ~/dsd/ghdl/*vhdl .
